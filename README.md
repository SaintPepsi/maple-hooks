# Maple Hooks

### A collection of claude code hooks to _hopefully_ make your AI Development Workflow more deterministic.

**Maple Hooks** is a collection of hooks designed to sit between you and your AI (specifically for **Claude Code**). These hooks are the "connective tissue" that helps your AI remember what it learned, stay within budget, and follow your coding standards without being asked.

---

## What can these hooks do?

Your imagination is the limit! Hooks can do ANYTHING; from simple things like reprompting on stop to continue working, to complicated workflow tracking where you need to make sure each file that's been touched has their automated tests ran upon completion

Some examples include:

**🧠 Memory & Learning:** Example: Automatically extracts "learning signals" from your conversations so your AI actually gets smarter over time.

**🛡️ Security & Safety:** Example: Blocking the AI from accidentally deleting your core settings or bypassing your security rules.

**⚖️ Quality Control:** Examples: Enforce strict TypeScript rules, update documentation obligations, and ensures your code adheres to quality you are happy with.

**📈 Session Analytics:** Example: Tracking exactly how much your prompts cost or how many times you've repeated the same prompt today

**🔄 Workflow Sync:** Example: Automatically keeps your PRDs, git branches, and project documentation in sync with the code.

## 🛠️ Quick Start

Ready to supercharge your setup? If you have **Bun** installed, it’s a one-liner:

```bash
git clone https://github.com/SaintPepsi/maple-hooks.git
cd maple-hooks
bun install && bun run install-hooks
```

_This will safely merge the hooks into your `settings.json` and keep your existing setup intact._
